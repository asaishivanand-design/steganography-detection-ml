# Image Steganography Detection using Machine Learning

## 📌 Overview

This project focuses on detecting hidden information in images using machine learning techniques. It compares multiple models to evaluate their effectiveness in identifying steganographic content.

## 🧠 Models Used

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)

## 📊 Results

* Logistic Regression: 83.33%
* Random Forest: 81.67%
* SVM: 83.33%

## ⚙️ Methodology

* Synthetic dataset generation
* Noise-based steganography simulation
* Feature extraction (statistical + edge + texture)
* Model training and evaluation
